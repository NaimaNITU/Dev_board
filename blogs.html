Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements using getElementById("id") for ID, getElementsByClassName("class") for class, getElementsByTagName("tag") for tag, querySelector("selector") for the first match, querySelectorAll("selector") for all matches, and getElementsByName("name") for elements by name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

`innerHTML` sets or gets HTML content with tags, `innerText` handles only visible text, and `textContent` retrieves all text, including hidden content, as plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling, improving performance and handling dynamic elements efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event triggered on an element propagates upwards through its ancestors (parent, grandparent, etc.) in the DOM hierarchy. This allows parent elements to listen for events on their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element: `let newElement = document.createElement("div");` To add an element: `document.body.appendChild(newElement);` To remove an element: `newElement.remove();` or `parent.removeChild(child);`
